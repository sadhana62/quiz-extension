Privacy Policy

Extension: Google Forms Study Helper
Effective date: March 24, 2026

Google Forms Study Helper is a browser extension that adds an "Explain with AI" button to multiple-choice questions on Google Forms. This page explains what data the extension handles and how that data is used.

What information the extension handles

  • Your Gemini API key, which you enter in the extension settings.
  • Question text and answer options from the Google Forms page when you click "Explain with AI".
  • Basic extension settings stored in browser storage.

How the information is used

  • Your Gemini API key is used only to authenticate requests to the Gemini API.
  • Question text and answer options are sent to the Gemini API only after you click the explanation button.
  • The AI response is displayed inside the page to help you study the question.

Where data is stored

The extension stores your Gemini API key in Chrome or Edge sync storage using the browser extension storage API. The extension does not operate its own backend server and does not store your data on a developer-controlled database.

Who data is shared with

When you request an explanation, the selected question content and your API key are transmitted directly to Google's Gemini API endpoint. No other third-party services are used by this extension based on the current codebase.

What the extension does not do

  • It does not collect account registration details.
  • It does not sell personal information.
  • It does not run analytics, advertising, or remote tracking scripts.
  • It does not send form content anywhere unless you actively request an explanation.

Data retention and control

You can remove the stored API key at any time by clearing the extension settings or uninstalling the extension. Data sent to the Gemini API is subject to Google's terms and policies for that service.
