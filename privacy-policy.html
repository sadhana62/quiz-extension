Privacy Policy

Extension: Google Forms Study Helper
Effective date: March 24, 2026

Google Forms Study Helper is a web app that lets a user submit a public Google Form URL and a Gemini API key to receive study-oriented explanations for multiple-choice questions. This page explains what data the app handles and how that data is used.

What information the extension handles

  • Your Gemini API key, which you enter into the web app form.
  • The Google Form URL you submit.
  • Question text and answer options extracted from the public Google Form.

How the information is used

  • Your Gemini API key is used only to authenticate requests to the Gemini API.
  • The submitted Google Form URL is fetched to read the public question content.
  • Question text and answer options are sent to the Gemini API to generate study-oriented analysis.
  • The AI response is displayed inside the page to help you study the question.

Where data is stored

The app does not intentionally store your Gemini API key in a developer-controlled database. Requests are processed through the hosted app and Google Gemini API in order to fetch the form and return the analysis.

Who data is shared with

When you request an analysis, the submitted form URL is fetched by the hosted app and the extracted question content plus your API key are sent to Google's Gemini API endpoint. No other third-party services are used by this app based on the current codebase.

What the extension does not do

  • It does not collect account registration details.
  • It does not sell personal information.
  • It does not run analytics, advertising, or remote tracking scripts.
  • It does not process Google Form content unless you actively submit a form URL.

Data retention and control

You can stop sharing data with the app at any time by closing the page and not submitting additional requests. Data sent to the Gemini API is subject to Google's terms and policies for that service.
